# Hit #2 — Pintando con código

---

En el video Painting a Landscape with Maths, Iñigo Quilez demuestra que es posible generar un paisaje completo utilizando únicamente código matemático dentro de un shader. En lugar de emplear imágenes o modelos prediseñados, cada píxel de la escena es calculado mediante funciones trigonométricas, interpolaciones, ruido procedural y operaciones de color.  

A través de estas fórmulas se construyen distintos elementos visuales como el cielo, las montañas, la niebla, el agua y la vegetación, logrando sensación de profundidad, iluminación y movimiento. Todo el paisaje surge entonces de evaluar expresiones matemáticas de manera continua sobre la pantalla, lo que evidencia el enorme potencial creativo de los fragment shaders y del renderizado procedural.  

La principal conclusión que obtuvimos es que las matemáticas no solo permiten resolver cálculos gráficos, sino también crear imágenes complejas y artísticas sin necesidad de recursos externos. Un shader bien diseñado puede transformar simples ecuaciones en escenas visualmente especiales, demostrando que en computación gráfica el código puede funcionar como una herramienta de pintura digital.
