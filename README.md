## Instalación

Clonar el repositorio:

```
git clone https://github.com/GonzaEC/SD-2026-ABC
cd SD-2026-ABC
```

Instalar dependencias:

```
pip install -r requirements.txt
```

## Pruebas

Para ejecutar las pruebas automatizadas:

```
pytest
```
