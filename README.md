## Instalación

Clonar el repositorio:

```
git clone <URL_DEL_REPOSITORIO>
cd <NOMBRE_DEL_REPO>
```

Instalar dependencias:

```
pip install -r requirements.txt
```

## Pruebas

Para ejecutar las pruebas automatizadas:

```
pytest
```
